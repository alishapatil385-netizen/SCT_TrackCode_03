# HR Analytics Dashboard 📊

## Project Overview
This Power BI dashboard analyzes employee attrition, demographics, salary distribution, and job role trends to help HR teams make data-driven decisions.

## Key Metrics
- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary
- Average Years at Company

## Visualizations
- Attrition by Education
- Attrition by Age Group
- Attrition by Salary Slab
- Attrition by Gender
- Attrition by Job Role

## Tools Used
- Power BI
- DAX
- Data Modeling

## Insights
- Highest attrition observed in Sales Executive role.
- Employees aged 26-35 show maximum attrition.
- Lower salary slabs have higher attrition rates.
